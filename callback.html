Note: GitHub OAuth requires a server to complete the authentication flow securely. Since Polymathica is a static site, we'll use Personal Access Tokens instead.

Create a Personal Access Token

  1. Go to GitHub Settings ‚Üí Developer settings ‚Üí Personal access tokens ‚Üí Tokens (classic)
  2. Click "Generate new token (classic)"
  3. Give it a note like: Polymathica Learning Tracker
  4. Set expiration to "No expiration" (or your preference)
  5. Select the "repo" scope (full control of private repositories)
  6. Click "Generate token"
  7. Copy the token (you won't be able to see it again!)

Save Your Token

Once you have your token, return to Polymathica and paste it in the authentication dialog.
